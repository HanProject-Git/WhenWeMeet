1. CORS 문제 해결

프론트(localhost:5173) → 백엔드(localhost:8080) 요청 시:

No 'Access-Control-Allow-Origin'

에러 발생.

해결:

Spring Security + CORS 설정 추가
credentials: "include" 유지
세션 기반 로그인 정상 동작

결과:

/api/auth/login
/api/auth/me

정상 통신 완료.

7. 회원탈퇴 기능 구현

추가 API:

DELETE /api/auth/me

구현 흐름:

회원
 → participant 삭제
   → available_date 삭제
 → member 삭제
 → session invalidate

해결한 문제:

foreign key constraint
participant 참조 중

참조 데이터 먼저 삭제하도록 수정.
