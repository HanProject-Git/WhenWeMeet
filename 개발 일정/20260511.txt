2. Docker PostgreSQL 구축
docker-compose.yml 생성

  postgres:
    image: postgres:16
실행
docker compose up -d
확인
docker ps

5. Room 기능 개발
Room Entity 생성

테이블:

room

컬럼:

id
title
description
startDate
endDate
inviteCode
createdAt
Room 생성 API
POST /api/rooms

요청:

{
  "title": "백엔드 회의",
  "description": "팀 일정 조율",
  "startDate": "2026-05-15",
  "endDate": "2026-05-20"
}
Room 조회 API
GET /api/rooms/{id}
InviteCode 조회 API
GET /api/rooms/invite/{inviteCode}
6. Participant 기능 개발
Participant Entity

관계:

Room 1 : N Participant

JPA:

@ManyToOne

적용.

참여자 등록 API
POST /api/rooms/{roomId}/participants

요청:

{
  "name": "김철수"
}
7. AvailableDate 기능 개발
AvailableDate Entity

관계:

Participant 1 : N AvailableDate
가능한 날짜 저장 API
POST /api/participants/{participantId}/available-dates

요청:

{
  "dates": [
    "2026-05-15",
    "2026-05-16",
    "2026-05-20"
  ]
}
8. 일정 계산 로직 구현
날짜별 가능 인원 계산
GET /api/rooms/{roomId}/result

응답:

[
  {
    "date": "2026-05-15",
    "count": 2
  }
]
모두 가능한 날짜 계산
GET /api/rooms/{roomId}/result/common

응답:

[
  "2026-05-15",
  "2026-05-20"
]

핵심 로직:

stream
groupingBy
counting

사용.
